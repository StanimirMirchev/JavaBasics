1.Стъпка
Имате име на папка,в която се съдържа .java файла ви.
команда: cd \ИмеНаПапка я прави текуща.
2.Стъпка
команда: dir показва съдържанието на текущата папка.
3.Стъпка
команда: set path=%path%;C:\Program files\Java\jdk...\bin
Забележка: на мястото на точки стой версията на вашето jdk.
Тази команда казва на системата къде да търси JDK програми.
4.Стъпка
команда: javac ИметоНаВашатаПрограма.java 
Това изпълнява javac.exe, компилаторът.Не трябва да ви се показва нищо.
5.Стъпка
Ако погледнете какво има в директория на вашия файл с командата от стъпка 2
ще видите,че командата от стъпка 4 е създала ИметоНаВашатаПрограма.class файл.
6.Стъпка
команда: java ИметоНаВашатаПрограма.java
Командата изпълнява Java интерпретатора.Трябва да видите изхода на вашата програма.
